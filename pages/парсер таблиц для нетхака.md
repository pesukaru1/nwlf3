### Задача:
	- Необходимо заполнить `.json` данными по такому шаблону:
	- ```TypeScript
	  interface Monster {
	    id: string;            // внутриигровой идентификатор
	    name: string;          // отображаемое имя
	    damage_range: { min: number; max: number };
	    resists: string[];     // ['cold', 'poison', ...]
	    speed: 'slower' | 'normal' | 'faster';
	    corpse: {
	      edible: boolean;
	      risks: string[];     // ['illness', 'poison', ...]
	      intrinsics: { resist: string; reliability: string }[];
	    };
	  }
	  ```
	- Чтобы на выходе иметь подобное:
	- ```json
	  {
	    "bat": {
	      "id":"PM_BAT",
	      "name":"bat",
	      "damage_range": {
	        "min":1,
	        "max":8
	      },
	      "resists":["cold",
	        "poison"],
	      "speed":"faster",
	      "corpse": {
	        "edible":true,
	        "risks":["illness"],
	        "intrinsics":[{
	          "resist":"cold",
	          "reliability":"common"
	        }]}}
	  }
	  ```
