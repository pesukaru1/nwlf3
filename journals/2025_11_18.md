- Для развёртывания вашей AI модели на сервере Timeweb Cloud, следуйте приведённой ниже пошаговой инструкции:
	- Выбор сервера:
		- Перейдите в панель управления [Timeweb Cloud](https://timeweb.cloud/my).
		- Выберите подходящий виртуальный сервер в зависимости от требований вашей модели (например, объём оперативной памяти, процессор и т.д.).
	- Загрузка файлов модели:
		- Подключитесь к серверу через SSH.
		- Загрузите файлы вашей модели на сервер, например, в директорию `/home/your_user/model`.
	- Создание виртуального окружения:
		- Установите Python, если он не установлен: `sudo apt install python3 python3-venv`.
		- Создайте виртуальное окружение: `python3 -m venv venv`.
		- Активируйте его: `source ./venv/bin/activate`.
	- Установка зависимостей:
		- Убедитесь, что у вас есть файл `requirements.txt` с необходимыми библиотеками.
		- Установите зависимости: `pip install -r requirements.txt`.
	- Настройка и запуск модели:
		- Если ваша модель использует Docker, следуйте [инструкции по деплою из Dockerfile](https://timeweb.cloud/docs/apps/deploying-with-dockerfile).
		- Для запуска модели в фоне используйте `screen`:
		  
		  ```
		  sudo apt install screen
		  screen
		  source ./venv/bin/activate
		  python your_model_script.py
		  ```
		- Сверните сессию `screen`: нажмите `Ctrl + A`, затем `D`.
	- Настройка автозапуска (опционально):
		- Настройте автозапуск вашего скрипта при перезагрузке сервера, используя системные службы или crontab.
		  
		  Если у вас возникнут вопросы или потребуется помощь, пожалуйста, свяжитесь с поддержкой по адресу info@timeweb.cloud.
